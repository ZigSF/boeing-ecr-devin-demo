# Boeing Engineering Change Request (ECR) Automation Demo

This repository demonstrates an autonomous engineering workflow agent that processes Engineering Change Requests (ECRs), classifies them, routes them to the appropriate engineering teams, generates compliance requirements, and creates tracking issues.

## Problem

Large aerospace organizations struggle with:
- Engineering backlog routing
- Manual compliance interpretation
- Cross-team coordination overhead
- Slow engineering change triage cycles

## Solution

This demo simulates an AI engineering operations agent that:
1. Classifies engineering change requests
2. Routes them to the correct engineering team
3. Generates compliance checklists based on safety impact
4. Creates a structured engineering tracking issue
